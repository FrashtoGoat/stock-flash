# Stock Flash

**A 股智能交易助手**：定时或命令触发，自动完成「新闻抓取 → 去重 → 预处理 → LLM 分析 → 链式筛选 → 大盘判断 → 通知 → 模拟交易」全流程。

---

## 第一性原理（项目指导思想）

用**第一性原理**指导写这个程序：从「识别利好 → 筛选 → 判断 → 通知 → 交易」的本质出发，**优先最快让想法落地**，而不是纠结前端好不好看、要不要换 Linux、架构是否完美。  

**快速落地 → 测试 → 发现问题 → 回测 → 持续正反馈**，是保持热情、不迷失方向的关键；时间一久、精力分散，容易忘了自己在干嘛，所以先跑通、再迭代。

## 流水线（8 步）与两条路线

```
定时/命令触发
      │
      ▼
┌─────────────────────────────────────────────────────────────────────────┐
│ Step 1  拉取新闻 (新浪 + 东方财富)                                        │
│ Step 2  去重 (新闻库 DB 或文件 news_id，超时标 expired)                   │
│ Step 3  预处理 → 分类/情绪/关键词 → 预筛选                               │
│ Step 4  LLM 分析 (利好 + 利空 并行) → 标的 + 大盘/行业利空                 │
│ Step 5  链式筛选 (板块→可买→异动→基本面→机构→技术)                       │
│ Step 6  大盘判断 (上证+参考指数，非交易日用最近交易日数据)                 │
│ Step 7  通知 (邮件 / 企微 / PushPlus；含买入与卖出信号)                    │
│ Step 8  交易执行 (模拟/模拟盘/CTP：result/trades/日期/ + DB)             │
└─────────────────────────────────────────────────────────────────────────┘
```

- **新闻驱动**（`python run.py`）：拉取→去重→预处理→LLM→链式筛选→通知→交易；通过筛选的标的入股票池。
- **自研池**（`python run.py --research-pool`）：从 `research_pool.stocks` 出发，只跑链式筛选→大盘判断→通知→交易，来源标明「自研池」。
- **止盈/止损**：流水线末尾检查持仓，若触发止盈或止损会**同样发通知并执行卖出**，与买入信号一致。

## 已实现功能（概要）

- 新闻库、复盘 `--review`、止盈止损、模拟/模拟盘/CTP、股票池（观察→稳定→高位→移除）、回测 `--backtest`、定时双路线调度 `--schedule`（8:00–01:00 每小时）。
- 通知内容包含：**信心说明**（为何是该百分比）、**满足条件**（原因中通过环节）、**建议买入价/建议卖出价**（链式筛选后按现价与配置计算）、**买入/卖出**均会通知。

## 快速开始

```bash
pip install -r requirements.txt
cp .env.example .env   # 编辑 .env：API Key、邮箱等
python run.py          # 执行一次流水线
python run.py --test   # 测试模式（石油新闻全流程）
python run.py --schedule   # 定时调度（双路线，8:00–01:00 每小时）
python run.py --backtest   # 回测
python run.py --review    # 复盘
python run.py --research-pool  # 自研池路线
python run.py --macro     # 拉取一次宏观快照（美联储利率、美债收益率等）
```

配置与扩展见 **config/settings.yaml** 与 [todolist_skill.md](todolist_skill.md)。
